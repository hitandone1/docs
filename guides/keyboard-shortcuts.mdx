Keyboard shortcuts let you move through Jetie at full speed without reaching for the mouse. The shortcuts below cover every major action in the app, organized into four categories. Most shortcuts follow familiar conventions from other productivity tools, so you'll find your muscle memory transfers quickly.

<Tip>
  You can customize any shortcut to fit your own workflow. Go to **Settings ΓÁÈ Shortcuts**, click the shortcut you want to change, and press your preferred key combination. Custom shortcuts are synced across all your devices.
</Tip>

## Navigation

Use these shortcuts to move between pages, open panels, and search your workspace without touching the sidebar.

| Action | Mac | Windows / Linux |
|---|---|---|
| Open command palette | `Cmd+K` | `Ctrl+K` |
| New page | `N` | `N` |
| Open AI Chat | `Cmd+Shift+A` | `Ctrl+Shift+A` |
| Focus sidebar | `Cmd+\` | `Ctrl+\` |
| Search workspace | `Cmd+P` | `Ctrl+P` |
| Go back | `Cmd+[` | `Alt+Left` |

## AI

These shortcuts give you fast access to Jetie's AI features ΓÇõ both the full chat panel and the inline suggestion layer inside the editor.

| Action | Mac | Windows / Linux |
|---|---|---|
| Open AI Chat | `Cmd+Shift+A` | `Ctrl+Shift+A` |
| Inline AI on selection | `Cmd+J` | `Ctrl+J` |
| Accept inline suggestion | `Tab` | `Tab` |
| Dismiss inline suggestion | `Esc` | `Esc` |

<Note>
  To trigger **Inline AI**, select any text in the editor and press `Cmd+J` (`Ctrl+J`). A small prompt bar appears directly below your selection ΓÇõ type an instruction like *"Rewrite this more concisely"* or *"Translate to Spanish"* and press `Enter`.
</Note>
